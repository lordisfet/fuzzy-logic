Для генерації документації локально, переконайтеся, що у вас встановлено Java 17+ та Maven.

1. Відкрийте термінал у кореневій директорії проєкту `fuzzy-logic`.

2. Виконайте команду: `mvn javadoc:javadoc`

3. Згенерована HTML-документація буде доступна у директорії: `target/site/apidocs/`

4. Відкийте файл `target/site/apidocs/index.html` у вашому браузері.
