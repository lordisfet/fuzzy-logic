# Fuzzy Logic System (Система нечіткої логіки)

## Опис проєкту
Цей репозиторій містить реалізацію системи нечіткої логіки (Fuzzy Logic) та інструменти для роботи з нечіткими множинами. Проєкт розроблено для виконання операцій фазифікації, застосування правил нечіткого висновку та дефазифікації.

## Основний функціонал
* Створення та налаштування нечітких множин.
* Базові операції над нечіткими множинами (перетин, об'єднання, доповнення).
* Налаштування лінгвістичних змінних та правил висновку.

## Запуск проєкту
1. Склонуйте репозиторій:
   `git clone https://github.com/lordisfet/fuzzy-logic.git`
2. Перейдіть у директорію проєкту:
   `cd fuzzy-logic`
3. Встановіть необхідні залежності.
4. Запустіть головний файл.
